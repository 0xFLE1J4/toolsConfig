## toolsConfig

This repository contains the configuration files for the tools I use on a daily basis, such as **tmux**, shell environments, and other productivity utilities.

Its purpose is to provide a clear overview of my personal setup, along with concise descriptions of each tool and how they are configured. The goal is not only to centralize my configurations, but also to make them easy to understand, reuse, and adapt.

Whether you're looking for inspiration, optimization ideas, or ready-to-use configurations, this repository offers a practical insight into a streamlined and efficient workflow.
